== Description ==
Social Share Image Plugin lets you create custom dynamic Open Graph Images.

== How to use? ==
1. Go to the "Social Share Image" tab in backend.
2. Select Image Background Colors
3. Save settings (This Will be the global image template; you can also change these setting for particular post in "Edit Post").
4. (For existing Posts) Go To Edit Post and Click "Update" to generate image for first time.

== Installation ==
1. Upload "social-share-image" to the "/wp-content/plugins/" directory using FTP or upload plugin zip using upload button.
2. Activate the plugin through the "Plugins" menu in WordPress.
